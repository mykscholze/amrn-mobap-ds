Follows the basic Google Styleguide for html
The following are general guidelines with naming and other conventions.
Classes: All lowercase
use '-' to denote separating words. e.g. "icon-select"
Spacing: To indicate nesting, use only two spaces. Using tabs can cause problem when be updated on the web.
File naming: Because we are working with sass, all html files should be named the same thing as their respective sass file.
All files should be in all lowercase.
Remember to be consistent with the rest of the code. If some conventions are not followed within a document or folder, stick with the whatever conventions are used.
