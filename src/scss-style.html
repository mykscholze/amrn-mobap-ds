• Classes:
    1. List @extends first.
    2. List @includes next.
    3. List "Regular" styling next. This would be tags like background color or width. These should also be in alphabetical order to help with compilation.
    4. Pseudo classes/elements next. These would be functions like "&:hover" or "&:before".
    5. Nested Selectors last. All items within the nested selector follows the same order.
